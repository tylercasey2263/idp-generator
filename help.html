Help & How-To
Everything you need to get the most out of Player IDP.
🚀
Getting Started
1
Set up your club Admin
Go to Settings in the sidebar. Add your club name, upload a logo, and paste your Anthropic API key. This only needs to be done once. Admins can also pick a theme colour for the whole portal.
2
Invite your coaches Admin
Go to Users in the sidebar. Click + Invite User, enter the coach's name and email, and select the Coach role. They'll receive an email with a sign-up link. Once they've joined, use the Manage Teams button on their user row to assign them to their teams.
3
Create your teams Coach
From the Dashboard, click + Add Team. Give it a name (e.g. "U14 Boys Blue") and an optional season label. You'll be automatically assigned to that team.
4
Add your players Coach
Click into a team and use + Add Player to add individually, or ⬆ Import CSV to bulk-upload your roster from a spreadsheet. Use ⇄ Import Player to pull a player from another team (they stay on both). All Players in the sidebar shows the full club roster.
5
Generate an IDP Coach
Click ✦ Generate IDP next to a player. Record or paste your coaching notes in the transcript — or click 🎙 to dictate directly. Hit Clean & Generate to have Claude AI build a professional development plan. Every section is editable inline before you publish. Each save creates a new version — previous IDPs are always preserved.
6
Invite parents Coach
Once you've published an IDP, go to the team page and click ⤴ Invite Parent next to the player. Enter the parent's email and hit Send — they'll receive an invitation email. When they accept, they're automatically linked to see all published IDPs for their child. Parents are also emailed automatically every time you publish a new IDP.
7
Generate a Team Development Plan Coach
On any team page, click ✦ Team Plan. The system pulls all published player IDPs, and Claude synthesises them into a team-wide plan: collective strengths, training themes, player spotlights, and a season vision. Add optional context notes before generating.

CSV Import Format
→
Supported format
Your CSV must have a Name column. A Position column is optional. The first row can be a header or data — the importer detects it automatically.
# With headers (recommended) Name,Position John Smith,Central Midfielder Jane Doe,Forward Alex Turner,Goalkeeper # Without headers also works John Smith,Central Midfielder Jane Doe,Forward
→
Exporting from Excel or Google Sheets
In Excel: File → Save As → CSV (Comma delimited)
In Google Sheets: File → Download → Comma-separated values (.csv)

Lineup Manager
→
What is it?
The Lineup Manager is a built-in soccer formation tool powered by Coach Assist. Use it to plan formations, assign players to positions, and prepare for matches — all from within the portal.
→
How to access it
Click Lineup Manager in the sidebar. Your session stays active — no separate login required.

Roles & Permissions
A
Admin
Full access to everything. Admins see all teams across the club, can manage any player or IDP, configure Settings (club name, logo, API key, theme), manage user roles, and assign coaches to teams. Typically the club director or head administrator.
C
Coach
Scoped to assigned teams only. Coaches can create teams, add/edit/remove players on their teams, generate and publish IDPs, log training sessions, run Team Plans, and invite parents. A coach cannot see or edit players on teams they're not assigned to. Admins assign teams via Users → Manage Teams; coaches can also self-link from the team page.
P
Parent
Read-only access to published IDPs for their linked child only. Parents join via an invite email sent by a coach. If a parent has multiple children in the club, they see a tab bar with each child's name. Parents can upload a photo for their child and send private feedback messages to the coach. They cannot access any other player's data.
→
Changing roles
Admins can change any user's role in Users (sidebar). Role changes take effect on the user's next sign-in. No user can promote their own role — this is enforced at the database level.

Frequently Asked Questions
Where do I get a Claude API key? ▼
Go to console.anthropic.com, sign in or create an account, navigate to API Keys, and create a new key. Paste it in Settings → AI Settings. You'll need to add billing credits to your Anthropic account for the key to work.
Can multiple coaches use the same account? ▼
No — each coach should have their own account. Have them sign up at the login page. An admin can then set their role to "Coach" in Settings → User Management.
How do parents access their child's IDP? ▼
Go to the team page, click ⤴ Invite Parent next to the player, and copy the unique invite link. Send it to the parent — they'll sign up and be automatically linked to see all published IDPs for that player.
Can I edit the IDP after generating it? ▼
Yes! On the IDP page, click the ✏ Edit button. All text sections become editable inline — click any text to modify it. When you're happy, click ⬆ Save & Publish to save and make it visible to the parent.
What if I have 50+ teams? ▼
Use the All Players page for a club-wide roster view. You can search by name, filter by team, edit player details, and reassign players between teams from one central location.
Is player data secure? ▼
Yes. All data is stored in a private Supabase database with row-level security enforced on every table. Coaches only see players and IDPs on their assigned teams. Parents only see published IDPs for their own linked child — no cross-family access is possible. The Claude API key is admin-only and cannot be read or changed by coaches via the browser.
How does IDP versioning work? ▼
Every time you publish an IDP, a brand-new version is created — previous versions are never overwritten. On the player's IDP history page you can see all versions newest-first, use the ⇄ What Changed diff panel to compare consecutive versions, and copy any version as a new draft. Parents always see the most recently published version on their portal tab.
Do parents get notified when a new IDP is published? ▼
Yes — if email notifications are configured. When you publish an IDP, the system automatically sends an email to any parent linked to that player. The email contains a direct link to the parent portal. This requires the notify-parent Edge Function to be deployed in Supabase and a Resend API key to be set in its secrets (see the manual setup instructions in TODO.md).
A parent has two children at the club — how does that work? ▼
Invite the parent separately for each child using ⤴ Invite Parent on each player's row (using the same parent email both times). Once accepted, the parent's portal shows a tab bar at the top with each child's name and photo. An amber ● New badge appears on a child's tab whenever a new IDP has been published since the parent last visited.
How do I assign a coach to a team? ▼
Two ways: (1) Admin path — go to Users in the sidebar, find the coach's row, and click Manage Teams. Check the teams you want to assign and uncheck any to remove. (2) Self-link — any coach can open a team page and click Link me to this team in the Coaching Staff section to add themselves.
