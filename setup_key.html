IDP Key Setup

This tool encrypts your Anthropic API key with the team password using AES-256-GCM. Open this file locally in your browser — nothing is sent anywhere. Run it once, then paste the blob into index.html.

Next step:
Open index.html, find this line near the top of the <script> block:

const ENCRYPTED_KEY_BLOB = '';

Replace the empty string with the blob above:

const ENCRYPTED_KEY_BLOB = '…paste here…';

Save, commit, and push. Coaches will be prompted for the team password when they visit the site. You can now delete setup_key.html from the repo (it's in .gitignore by default).
